# Problem 0 — Sum of Two Integers

Read two integers and output their sum.

## Input Format

Two integers separated by whitespace.
They may appear:
- On the same line, or
- On separate lines.

## Output Format

A single integer: the sum of the two integers.

## Example

Input:
3 4

Output:
7

## Constraints

- Integers fit in 64-bit signed range.

## Expected Complexity

- Time: O(1)
- Space: O(1)
